# 旅途記帳 PWA

一個功能完整的旅遊記帳應用程式，採用 Progressive Web App (PWA) 技術開發，支援離線使用。

## 功能特性

- **旅程管理** — 建立和管理多個旅程
- **記帳功能** — 記錄旅途中的各項支出
- **景點追蹤** — 記錄訪問的景點和時間
- **行李清單** — 管理旅行必需品清單
- **成員分帳** — 多人旅行時的費用分配
- **統計分析** — 支出分類統計和分析
- **離線支援** — PWA 技術支援離線使用
- **響應式設計** — 完美適配各種裝置

## 技術堆疊

- **前端框架** — 原生 HTML5 + CSS3 + JavaScript
- **PWA 技術** — Service Worker + Web App Manifest
- **本地存儲** — LocalStorage + IndexedDB
- **響應式設計** — Mobile-first 設計方法

## 快速開始

### 本地運行

1. 克隆本倉庫：
```bash
git clone https://github.com/your-username/旅途記帳_PWA.git
cd 旅途記帳_PWA
```

2. 使用本地伺服器運行（推薦使用 HTTPS）：
```bash
# 使用 Python 3
python3 -m http.server 8000

# 或使用 Node.js
npx http-server
```

3. 在瀏覽器中打開：
```
http://localhost:8000
```

### 安裝為 PWA

1. 在支援的瀏覽器中打開應用程式
2. 點擊地址欄的「安裝」按鈕或使用選單中的「安裝應用程式」選項
3. 應用程式將安裝到您的裝置主螢幕

## 使用說明

### 建立新旅程

1. 點擊首頁的「新增旅程」按鈕
2. 輸入旅程名稱、開始日期和結束日期
3. 點擊「建立」完成

### 記錄支出

1. 進入旅程詳情頁面
2. 點擊浮動按鈕「+」新增支出
3. 選擇分類、輸入金額和備註
4. 點擊「保存」完成

### 管理景點

1. 在旅程詳情頁面選擇「景點」標籤
2. 點擊「+」新增景點
3. 輸入景點名稱和訪問時間
4. 點擊「保存」完成

### 分帳管理

1. 進入旅程詳情頁面
2. 點擊「分帳」標籤
3. 系統會自動計算各成員應付或應收金額

## 離線功能

本應用程式使用 Service Worker 技術支援離線使用：

- 首次訪問後，應用程式將被快取
- 在無網路連線時，仍可正常使用所有功能
- 資料儲存在本地裝置，不依賴網路連線

## 資料隱私

- 所有資料均儲存在本地裝置
- 不會上傳到任何伺服器
- 完全保護您的隱私
